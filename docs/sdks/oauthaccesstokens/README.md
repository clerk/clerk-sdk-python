## verify

Verify an OAuth Access Token

### Example Usage

```python
from clerk_backend_api import Clerk


with Clerk(
    bearer_auth="<YOUR_BEARER_TOKEN_HERE>",
) as clerk:

    res = clerk.oauth_access_tokens.verify(access_token="XXXXXXXXXXXXXX")

    assert res is not None

    # Handle response
    print(res)

```
