## list

Lists all whitelisted redirect_urls for the instance

### Example Usage

```python
from clerk_backend_api import Clerk


with Clerk(
    bearer_auth="<YOUR_BEARER_TOKEN_HERE>",
) as clerk:

    res = clerk.redirect_urls.list(paginated=False)

    assert res is not None

    # Handle response
    print(res)

```
